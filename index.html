How to sync memories

This tree reads from a GitHub Gist. When Claude learns something new about you, sync in 3 steps:

1
In any Claude chat, type: sync my memory tree
Claude will output a fresh memories.json
2
Go to your Gist → click Edit → paste the new JSON → click Update public gist
3
Refresh this page — the tree will load fresh data automatically from your Gist

Also commit memories.json to your repo as a backup anytime.
