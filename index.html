How to sync memories

When Claude learns something new about you, update the tree in 3 steps:

1
In any Claude chat, type: sync my memory tree — Claude outputs fresh JSON data
2
Open index.html → find the MEMORY_DATA object → replace it with the new JSON Claude gives you
3
Commit & push to GitHub → page auto-updates with latest memories

No Gist, no CORS, no loading issues — data lives in the file itself.
